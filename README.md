## GIS & Spatial Analysis

Tools for geospatial data processing, watershed delineation, and spatial hydrological analysis.

| Resource | Description |
|---|---|
| [QGIS](https://www.qgis.org/) | Free and open-source desktop GIS application with extensive hydrology plugins. |
| [GRASS GIS](https://grass.osgeo.org/) | Powerful GIS suite with built-in tools for terrain analysis and watershed modeling. |
| [WhiteboxTools](https://www.whiteboxgeo.com/geospatial-software/whitebox-tools/) | Advanced geospatial analysis platform with 500+ tools including hydrology-specific functions. |
| [SAGA GIS](https://saga-gis.sourceforge.io/) | System for Automated Geoscientific Analyses with strong terrain/hydrology modules. |
| [TauDEM](https://hydrology.usu.edu/taudem/taudem5/) | Terrain Analysis Using Digital Elevation Models for watershed delineation. |

## Hydraulic Modeling Software

Software for simulating stormwater, sewer systems, water distribution networks, and open-channel flow.

| Resource | Description |
|---|---|
| [EPA SWMM](https://www.epa.gov/water-research/storm-water-management-model-swmm) | EPA's Storm Water Management Model for urban drainage and runoff simulation. |
| [EPANET](https://www.epa.gov/water-research/epanet) | EPA's water distribution system modeling software for pressure and quality analysis. |
| [HEC-RAS](https://www.hec.usace.army.mil/software/hec-ras/) | USACE river analysis system for 1D/2D steady and unsteady flow simulation. |
| [HEC-HMS](https://www.hec.usace.army.mil/software/hec-hms/) | USACE hydrologic modeling system for rainfall-runoff simulation. |
| [MIKE URBAN+](https://www.dhigroup.com/technologies/mikepoweredbydhi/mike-urban-plus) | DHI's integrated urban water modeling platform (free academic license available). |
| [OpenFOAM](https://www.openfoam.com/) | Open-source CFD toolbox applicable to complex free-surface hydraulics. |

## Public Datasets

Freely accessible hydrological, meteorological, and geospatial datasets for research and modeling.

| Resource | Description |
|---|---|
| [USGS Water Data](https://waterdata.usgs.gov/nwis) | National real-time and historical streamflow, groundwater, and water-quality data. |
| [NOAA Climate Data Online](https://www.ncdc.noaa.gov/cdo-web/) | Precipitation, temperature, and storm event records from NOAA. |
| [Global Runoff Data Centre (GRDC)](https://www.bafg.de/GRDC/) | International repository of river discharge and runoff data. |
| [HydroSHEDS](https://www.hydrosheds.org/) | Hydrographic data and maps for global watersheds derived from SRTM elevation data. |
| [SRTM Digital Elevation Data](https://www2.jpl.nasa.gov/srtm/) | NASA's near-global digital elevation model at 30 m resolution. |
| [OpenTopography](https://opentopography.org/) | High-resolution topography data portal including LiDAR datasets. |
| [ERA5 Reanalysis](https://cds.climate.copernicus.eu/datasets/reanalysis-era5-single-levels) | ECMWF global atmospheric reanalysis dataset for climate and hydrology studies. |

## Python Libraries

Python packages for hydrological data retrieval, analysis, and modeling workflows.

| Resource | Description |
|---|---|
| [geopandas](https://geopandas.org/) | Extends pandas to support geospatial operations on vector data. |
| [rasterio](https://rasterio.readthedocs.io/) | Reads and writes geospatial raster data (GeoTIFF, NetCDF, etc.). |
| [pysheds](https://github.com/mdbartos/pysheds) | Lightweight watershed delineation and terrain analysis in Python. |
| [hydrofunctions](https://hydrofunctions.readthedocs.io/) | Easy access to USGS NWIS water data from Python. |
| [dataretrieval](https://doi-usgs.github.io/dataretrieval-python/) | USGS package for retrieving water data from NWIS and Water Quality Portal. |
| [swmmio](https://github.com/aerispaha/swmmio) | Tools for reading, analyzing, and visualizing EPA SWMM models in Python. |
| [WNTR](https://wntr.readthedocs.io/) | Water Network Tool for Resilience — works with EPANET models in Python. |
| [xarray](https://docs.xarray.dev/) | N-dimensional labelled arrays ideal for gridded climate/hydrology datasets. |
| [MetSim](https://metsim.readthedocs.io/) | Meteorological simulator and data disaggregation tool for hydrological modeling. |
